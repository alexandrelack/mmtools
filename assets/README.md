# Assets

Coloque sua logomarca nesta pasta com um destes nomes:

- `logo.png`
- `logo.svg`
- `logo.jpg`
- `logo.jpeg`
- `logo.webp`

O app carrega a logo automaticamente no topo da interface e na sidebar.
